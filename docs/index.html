Open-source TeX → PDF workflow

TEX to PDF Builder

Minimalna aplikacja do przekształcania plików .tex i projektów LaTeX .zip w PDF. Lokalnie działa jako FastAPI + Docker + LaTeX, a darmowe demo buduje PDF przez GitHub Actions.

View on GitHub Build PDF with Actions

Local Web App

Uruchamiasz lokalnie, wrzucasz .tex albo .zip i dostajesz PDF w przeglądarce.

uvicorn app.main:app --reload --host 127.0.0.1 --port 8000

Slayer Paper Mode

Formularz generuje strukturę raportu: main.tex, formatka/slayer.sty, tresc/00-strona-tytulowa.tex i tresc/10-tresc.tex.

Free Demo

Bez VPS i bez Rendera: GitHub Actions kompiluje przykładowy projekt LaTeX i publikuje PDF jako artifact.

Wejdź w Actions → Build Sample PDF → Run workflow.

Jak działa darmowy build PDF?

  1. W repo znajduje się przykładowy projekt w examples/sample-paper/.
  2. Workflow GitHub Actions instaluje LaTeX.
  3. Uruchamia latexmk.
  4. Publikuje main.pdf jako artifact.
